Project Overview

This project focuses on Banking Data Analysis using Python. The goal is to analyze customer and financial data to extract meaningful insights that can support business decision-making in the banking sector.

The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and insight generation.


Technologies & Libraries Used

Python 

Pandas – Data manipulation & analysis

NumPy – Numerical operations

Matplotlib – Data visualization

Seaborn – Statistical visualization

Jupyter Notebook – Interactive analys
